# Text Cleaner

Minimal web app to **clean messy text** instantly.
No signup.
No API.
Just paste and clean.

---

## Live Demo

https://text-cleaner-orpin.vercel.app/

---

## Features

- **Trim lines** – remove leading/trailing spaces from each line
- **Remove extra spaces** – collapse multiple spaces into one
- **Remove empty lines** – delete blank lines
- **Remove line breaks** – join all lines into a single paragraph
- **Case transforms:**
  - lowercase
  - UPPERCASE
  - Title Case
  - Sentence case
- **Live stats:**
  - Character count (input vs output)
  - Word count
  - Line count
- **One-click copy** – copy cleaned text to clipboard
- **Example text** – load sample text to test
- **Real-time preview** – see cleaned output as you type
- Clean, responsive, Apple-inspired dark UI
- No authentication
- No ads
- No watermarks

---

## Tech Stack

- React (Vite)
- Vanilla CSS (Apple-inspired UI)
- Vercel (static deploy)

---

## How It Works

1. **Paste your text** into the input area
2. **Toggle cleaning options** (trim, spaces, lines, case)
3. **See the result** instantly in the output panel
4. **Copy** the cleaned text with one click

> Everything is processed client-side. Nothing is uploaded.

---

## Privacy

All processing happens **locally in your browser**.
No data is sent to any server.
